S&P600 Sector Strength Comparison

This web application, powered by Python, conducts in-depth analyses of S&P600 companies' financial health. Utilizing HTML scraping, it extracts pertinent data from Wikipedia to compile a comprehensive list of S&P600 stocks. Using the Yahoo Finance library, it then retrieves and integrates the balance sheets of these companies into a structured pandas DataFrame. Subsequently, employing pandas' robust data manipulation capabilities, the application organizes the data according to the Global Industry Classification Standard (GICS) sectors. Finally, utilizing Matplotlib, it presents insightful visualizations, offering an overview of certain financial trends within the S&P600 index.

The graph below illustrates Gardiner's Ratio (Invested Capital to Total Debt Ratio) across the 6 GICS Sectors. Notably, the Tech Industry displays lower debt levels compared to Industrials and Finance. This observation aligns with the differing performance of indices such as the tech-heavy Nasdaq 100 Index and the Dow Jones Industrial Average.
